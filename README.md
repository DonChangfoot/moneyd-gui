# Moneyd GUI
> A GUI for Moneyd

- [Usage](#usage)
- [TODOs](#todos)

## Usage

```sh
git clone git@github.com:sharafian/moneyd-gui.git
cd moneyd-gui
npm install
npm run build
npm start
```

Then go to [localhost:7770](http://localhost:7770) in your browser.
